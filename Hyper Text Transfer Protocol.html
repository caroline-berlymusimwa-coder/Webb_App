Hyper Text Transfer Protocol (HTTP)

HTTP is the application protocol for distributed hypermedia systems, evolving through versions for speed and security.


Versions and Features

HTTP/1.1: Persistent connections. HTTP/2: Binary, multiplexing. HTTP/3: QUIC (UDP-based), 0-RTT resumption, multipath for mobile.


Headers Deep Dive

  • Cache-Control: max-age, no-cache directives override defaults.
  • ETag/Last-Modified: Conditional validation.
  • CORS: Preflight OPTIONS for cross-origin.
  • connection: Keep-Alive for reuse.
  • Others: Proxy-Authenticate, Transfer-Encoding.

Caching Strategies

Browser (localStorage), proxy (shared), gateway (reverse proxy). Directives control staleness; transparent proxies preserve auth headers.
