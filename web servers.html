Web servers

Web servers manage incoming requests, serving static/dynamic content efficiently under high loads.

Request-Response Cycle

Persistent connections (HTTP/1.1 Keep-Alive); parse URI, headers, auth (Basic/OAuth). Static: file system reads. Dynamic: CGI → PHP/Node/Python → DB queries (MySQL/PostgreSQL). Reverse proxies (Nginx) cache/balance.

Architectures

  • Thread-per-request (Apache mpm_worker).
  • Event-driven (Nginx, Node.js). Serverless (AWS Lambda) auto-scales.

Popular Software

Apache (60% sites), Nginx (35%), LiteSpeed. Config: .htaccess, virtual hosts, mod_rewrite.

Monitoring and Optimization

Logs (access/error), tools (New Relic). HTTPS mandatory (Let's Encrypt free certs).
