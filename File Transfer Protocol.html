File Transfer Protocol

FTP is a client-server protocol (RFC 959) for bidirectional file transfers over TCP ports 20/21.


Modes and Commands

Transmission: Stream (default, TCP fragments), Block (headered), Compressed (bandwidth-saving). Commands: RETR (get), STOR (put), LIST/MLSD (listings), ASCII/binary
